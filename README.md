# Brute-Force

-----
## 완전 탐색이란?
* 문제를 해결하기 위해 확인해야 하는 모든 경우를 전부 탐색하는 방법   
* 백 트래킹 기법을 사용하여 문제를 해결   
* 경우의 수를 전부 탐색하기 때문에 시간 복잡도 측면에서 불리
